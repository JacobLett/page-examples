This project demonstrates the typical code you would need to build a Bootstrap based responsive website without.

Pros

  • Improves search engine optimization because Google will soon move to a mobile first index
  • Better user experience
  • Saves time developing common components from scratch
  • Has great documentation and design system to follow
Button

Cons

  • Includes code for components you may not use
  • Requires style overrides and a lot of style inspecting in Chrome DevTools
  • A learning curve. Need to follow a system and naming convention you didn't write yourself.
