This project demonstrates the typical code you would need to build a fixed width website without a frontend framework like Bootstrap.

Pros

  • Less time spent testing different browser widths
  • Control of how everything works and looks
Button

Cons

  • Requires more advanced knowledge of CSS and browser support of CSS3 properties
  • Takes a lot more time to customize because everything needs to be written and tested from scratch
  • No viewable documentation of basic components for team members. Or if you revisit the project after 6 months you don't have to recall your thought process.
  • Bad user experience
  • Poor search engine optimization because Google will soon move to a mobile first index
